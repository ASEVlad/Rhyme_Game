# Римова Гра

Українська веб-версія The Rhyme Game (Classic mode). Біт грає, м'ячик стрибає по сітці, у кожному рядку справа — слово, на яке має закінчитись твій бар.

## Local dev

```bash
npm install
cp .env.example .env.local
# fill in APP_PASSWORD, AUTH_SECRET (openssl rand -hex 32), ANTHROPIC_API_KEY
npm run dev
```

The repo ships with one placeholder beat (`public/beats/click-90.wav`, generated by `scripts/gen-beat.mjs`). Drop additional royalty-free `.mp3` files into `public/beats/` and add their metadata to `lib/beats.ts`. The `bpm` field must be derived from each file's actual duration:

```
bpm = barsPerLoop * 4 * 60 / audioDurationSeconds
```

## Deploy (Vercel)

1. Push to GitHub.
2. Import the repo in Vercel.
3. Set env vars: `APP_PASSWORD`, `AUTH_SECRET`, `ANTHROPIC_API_KEY`.
4. Deploy.
